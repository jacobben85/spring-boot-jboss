Docker Demo Application

The application is built to display capabilities of docker.

Docker (Docker compose) is used to install the following,

  • Docker UI, Docker UI gives insight on the docker containers running and their status.
  • Elasticsearch, ES gets installed standalone in a docker container with 9200 and 9300 ports exposed to the host.
  • Java application, A spring boot application gets pulled from github and is installed onto a container with 8080 port exposed to the host.
